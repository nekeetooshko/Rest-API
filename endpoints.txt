Авторизация/аутентификация:
    - POST /AUTH/SIGN-UP - регистрация
    - POST /AUTH/SIGN-IN - авторизация

Взаимодействие со списками:
    - GET /LISTS - вернет все
    - GET /LISTS/{ID} - вернет конкретный 
    - POST /LISTS - добавил 
    - PUT /LISTS/{ID} - обновит конкретный
    - DELETE /LISTS/{ID} - удалит конкретный
    - GET /LISTS/{ID}/ITEMS - вернет список айтемов конкретного элемента (видимо)
    - POST /LISTS/{ID}/ITEMS - добавит айтемы конкретному элементу (видимо)

Взаимодействие с элементами списка:
    - PUT /ITEMS/{ID} - обновит элемент
    - GET /ITEMS/{ID} - вернет конкретный элемент
    - DELETE /ITEMS/{ID} - удалит элементы конкретного списка
